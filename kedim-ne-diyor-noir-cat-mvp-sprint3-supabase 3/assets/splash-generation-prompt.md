# Noir Cat Splash Prompt

Vertical mobile splash screen, black gradient background, centered golden cat/paw/sound wave emblem, minimal premium Noir Cat aesthetic, high contrast, elegant soft gold glow, calm luxury mood, generous empty space for app title area, clean composition, no clutter, no realistic photo, mobile portrait format.

- Çok parlak renkler kullanılmasın.
- Gerçekçi kedi fotoğrafı kullanılmasın.
- Küçük ve okunmayan detaylar olmasın.
